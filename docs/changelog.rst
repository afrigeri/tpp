Changelog
=========

The source changelog is maintained in ``CHANGELOG.md`` at the project root.

Recent documentation-relevant changes include:

* shared result dictionaries now report upward-pointing unit normals;
* residual statistics are reported explicitly as orthogonal and vertical values;
* PCA/SVD uses the same residual-statistics fields as least squares and RANSAC;
* CSV exports record the fitting method stored in the computed result.
